
## 概述

Armx for Typecho 主題簡潔明了，一眼就看上了，但套用到自己站上有點問題，故做了定制化修改。

主要新增功能有：

1. 自適應圖庫，放大縮小播放隨心所欲；
2. 語音朗讀文章，用聲音感受世界；
3. 自動夜間模式，保護妳的眼睛；
4. 評論表情，心情躍然紙上；
5. 懶加載，提高資源利用；
6. 回復可見，精彩無限；
7. 獨立模板，豐富充實；
8. 高度自定義模塊，放飛個性；
9. ......
